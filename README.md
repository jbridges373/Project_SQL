# Project_SQL: Parks and Recreation Database

# SQL Portfolio: From Basics to Advanced Projects

Welcome to my SQL repository! This project serves as a comprehensive pipeline showcasing my SQL journey, tracking my progression from foundational queries to advanced database programming and full-scale portfolio data projects.

## 🚀 Repository Roadmap

### 🏁 Beginner Concepts
* **Parks_and_Rec_Create_db:** Database schema creation and mock dataset setup.
* **Select Statement:** Basic data retrieval and column selection syntax.
* **Where Statement:** Row filtering using logical operators and conditional logic.
* **Limit and Aliasing:** Result set constraints and renaming tables/columns.
* **Group By + Order By:** Aggregating metrics and sorting output systematically.
* **Having vs Where:** Understanding the exact operational differences in filtering aggregate data.

### 🔄 Intermediate Mastery
* **Joins:** Combining datasets via Inner, Left, Right, and Full Outer Joins.
* **Unions:** Stacking distinct result sets vertically using UNION and UNION ALL.
* **Case Statements:** Introducing conditional logic directly into SQL queries.
* **String Functions:** Cleaning and manipulating text strings (e.g., TRIM, SUBSTRING, REPLACE).
* **Subqueries:** Nesting queries inside SELECT, FROM, and WHERE clauses.
* **Window Functions:** Performing advanced calculations across a set of table rows (OVER, PARTITION BY).

### ⚡ Advanced Database Programming
* **CTEs (Common Table Expressions):** Writing cleaner, modular queries using defined temporary result sets.
* **Temp Tables:** Storing mid-session datasets to optimize query performance.
* **Stored Procedures:** Saving reusable SQL scripts to execute complex routines on command.
* **Triggers and Events:** Automating database updates based on specific actions or recurring schedules.

### 📊 Portfolio Projects
* **Data Cleaning:** Transforming raw data into a reliable, analytical-ready state.
* **EDA (Exploratory Data Analysis):** Querying historical datasets to uncover trends, distributions, and actionable insights.
